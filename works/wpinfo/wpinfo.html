Project Overview


WhitePhosphorus.info is a website aiming to document and analyze various atmospheric phenomena, particularly focusing on white phosphorus attacks in South Lebanon. This website has aggregated, transcribed, and developed a comprehensive, searchable, and interactive database to make the white phosphorus attacks in South Lebanon to be documented.


My Role


I was commissioned to design the web interface and implement the features with code. The project favors scientific, clear visual languages, as well as aesthetic values.

The web contains 3 sections: Geographic Space, Cyberspace, and Personal Space.

Project Goals:
1. Geographic Space: embeding location data of white phosphorus attacks and visualize the map with ArcGIS.
2. Cyberspace: allow real-time updates and displays of tweets on the map and as a graph.
3. Personal Space: displaying details of the Tweets that report on the topic.  

This is an ongoing project that started from May 2024. Wireframe and UI design was done with Figma, and data visualization is made possible by ArcGIS, D3, and React.js.
